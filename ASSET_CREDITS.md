# Asset Credits

## External Assets

### Blank Map of the United States

- **Source:** Wikimedia Commons, `Blank_US_Map.svg`
- **URL:** https://commons.wikimedia.org/wiki/File:Blank_US_Map.svg
- **License:** Public domain
- **Use:** Interactive campaign map and election-results map
- **Modifications:** State paths are normalized at runtime, recolored, textured through CSS, and given election metadata.

## Evaluated But Not Included

The planned CC0 UI, map-tile, ocean, city-prop, font, and industrial packs were evaluated but not included because their visual styles did not combine into a consistent interface. The game uses one public-domain map and a unified original interface instead.
